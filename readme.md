# Установка
Debian/Ubuntu
``` sudo apt install gcc nodejs npm ``` 
Arch
``` sudo pacman -S gcc nodejs npm ```
Установка шрифтов (JetBrainsMono Nerd Font)
``` sudo pacman -S ttf-jetbrains-mono-nerd ```
Установка плагинов
